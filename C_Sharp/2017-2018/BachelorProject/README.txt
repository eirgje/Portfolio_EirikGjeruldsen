-Nordavind- 

Using the Unity engine we created the game Nordavind as our bachelor project.
The game can be found here: https://neon-peon.itch.io/nordavind.

Due to this being a small team I had to do a little bit of everything.
This resulted in some of my time going into using systems already implemented in Unity's engine, which resulted in some of the code sheets having smaller amounts of code.
